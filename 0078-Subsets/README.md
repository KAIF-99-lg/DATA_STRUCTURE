# Subsets (LeetCode 78)

## Problem Statement
Given an integer array `nums` of **unique** elements, return **all possible subsets** (the power set).
The solution set **must not contain duplicate subsets**.

### Example 1:
**Input:**  
```plaintext
nums = [1,2,3]
```
**Output:**  
```plaintext
[[], [1], [2], [3], [1,2], [1,3], [2,3], [1,2,3]]
```

### Constraints:
- `1 <= nums.length <= 10`
- `-10 <= nums[i] <= 10`
- All elements of `nums` are **unique**.
